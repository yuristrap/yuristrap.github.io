# 디스플레이 (display)
디스플레이 속성을 변경할때 사용합니다.

<div class="card">

```css
.dsp-none {
	display: none !important;
}
.dsp-block {
	display: block !important;
}
.dsp-inline {
	display: inline !important;
}
.dsp-inline-block {
	display: inline-block !important;
}
.dsp-table {
	display: table !important;
}
.dsp-flex {
	display: flex !important;
}
```
</div>
